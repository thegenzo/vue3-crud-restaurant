# Vue JS 3 CRUD Restaurant with json-server

## Project setup
```
npm install
```

### Run json-server
```
json-server --watch data/db.json
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
